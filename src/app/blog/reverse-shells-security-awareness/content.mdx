## ğŸ”„ What Is a Reverse Shell?

A **shell** is simply a command-line interface â€” a way to execute commands on a system.

A **reverse shell** is when a target machine initiates a connection outward to an attacker-controlled system, giving the attacker remote command access.

**In a normal (bind) shell:**
- Attacker connects to the victim.

**In a reverse shell:**
- Victim connects to the attacker.

**This direction change is the key.**
