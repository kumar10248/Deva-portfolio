<CodeWindow title="Dockerfile">
```dockerfile

FROM node:18-alpine

# Set the working directory inside the container

# Copy package files first (for better caching)
COPY package*.json ./

# Install dependencies

# Copy the rest of the application
COPY . .

# Expose port 3000
EXPOSE 3000

# Command to run the application
CMD ["node", "index.js"]
```
</CodeWindow>
