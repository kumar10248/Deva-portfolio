## ğŸ” What is GPG?

**GPG (GnuPG)** is an open-source encryption tool based on the OpenPGP standard.

It lets you:

- Encrypt files
- Decrypt files
- Sign documents
- Generate key pairs
- Verify authenticity

It is widely used for:

- Secure file sharing
- Package verification
- Git commit signing
- Secrets management
- Email encryption

**In short, GPG = military-grade encryption built into Linux for free.**
